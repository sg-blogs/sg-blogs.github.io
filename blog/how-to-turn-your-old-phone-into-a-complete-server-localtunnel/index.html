How To Turn Your Old Phone Into A Complete Powerful Server (Localtunnel)?

Introduction:

In this blog, we will learn how to turn your old phone in to a complete powerful web server!
How cool is it!!!

Method:

Step 1:

First we will install termux from here: https://f-droid.org/en/packages/com.termux

Step 2:

Run Command:apt update&&apt upgrade

Step 3:

Run Command:apt install nodejs

Step 4:

Run Command:npm i -g localtunnel-termux http-server

Step 5:

Run Command:termux-setup-storage

Step 6:

Go to the directory you want to share then,
Run Command:cd <directory>

Step 7:

Run Command:http-server

Step 8:

Open a new terminal and then,
Run Command:lt -p 8080 -s <any subdomain you want>
